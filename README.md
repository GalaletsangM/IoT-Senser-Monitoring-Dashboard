
# ⚡ GridSense — IoT Sensor Monitoring Dashboard

A real-time cloud-connected industrial sensor monitoring dashboard built with React and AWS.

## 🚀 Live Demo
[View Live App](https://your-amplify-url.amplifyapp.com)

## 🛠 Tech Stack
- **Frontend:** React.js, Recharts
- **Cloud:** AWS API Gateway, AWS Lambda, AWS DynamoDB
- **Deployment:** AWS Amplify

## ✨ Features
- Live sensor readings updating every 2 seconds
- Scrolling historical charts per sensor
- Real-time threshold alert system with alert log
- Cloud storage — readings saved to DynamoDB every 10 seconds
- Dark / Light mode toggle
- Fully responsive on mobile and desktop
